[Tailwind CSS](https://tailwindcss.com/)
[Alpine.js](https://github.com/alpinejs/alpine)

During the development (for the watch mode) use:

```shell
npm start
```

For deployment, run:

```shell
npm run deploy
```

The output web page is at `dist` folder.
